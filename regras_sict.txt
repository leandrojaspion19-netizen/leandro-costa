REGRAS DE ACESSO E PERMISSÕES
GRUPO ADM (Edson, Fatima, Luiz): Responsáveis pelo cadastro inicial no campo NOVO PROCEDIMENTO. O botão [SALVAR] deve ser habilitado apenas com os dados básicos (Nome, Tipo de Doc, Data). O sistema está proibido de exigir campos técnicos (violação/agente) para o ADM.

CONSELHEIROS (Leandro, Luiza, Milena, Mirian, Sandra): Responsáveis pela triagem e aplicação de medidas.

2. LÓGICA DE DISTRIBUIÇÃO (Obrigatória)
CONSELHEIRO REFERÊNCIA: Se a família já existe, o sistema mantém o vínculo histórico. Se for nova, distribui sequencialmente entre os 5 conselheiros.

PROVIDÊNCIA IMEDIATA: Distribuição sequencial entre os 3 conselheiros escalados para o dia. Ao abrir o documento em "Minha Referência", o sistema deve:

Reconhecer o usuário logado e colocá-lo no topo da lista.

Abrir o documento em MODO DE EDIÇÃO (campos "Violação" e "Agente Violador" habilitados).

3. VALIDAÇÃO COLEGIADA (Segurança Jurídica)
ALERTA: Ao salvar uma medida, os outros 2 colegas do dia recebem um alerta imediato.

AÇÃO: O alerta deve ser um link direto (âncora) para a área de assinatura.

ASSINATURA: O botão [VALIDAR] deve executar uma gravação imediata no banco de dados. O documento só é finalizado com 2 validações extras.

PROMPT DE CORREÇÃO: RODÍZIO EQUITATIVO E AUTOMÁTICO
DIRETRIZ DE JUSTIÇA OPERACIONAL: O sistema deve interromper qualquer atribuição fixa. A distribuição deve ser rigorosamente igualitária entre os 5 conselheiros, respeitando o ciclo de disponibilidade.

1. LÓGICA DO CARROSSEL (RODÍZIO)
Distribuição por Documento: Para cada novo documento salvo pelo ADM, o sistema deve olhar quem foi o último conselheiro a receber um caso e atribuir o próximo da lista (Leandro -> Luiza -> Milena -> Mirian -> Sandra -> retorna ao Leandro).

Independência de Escolha: O ADM (Edson, Fatima, Luiz) não tem permissão para selecionar o nome. Ao salvar, o sistema faz o "check-in" do documento no próximo nome da fila de forma oculta.

2. CRUZAMENTO COM A ESCALA DE IMEDIATA
O sistema deve garantir que os 3 conselheiros do dia (Escala de Imediata) recebam volumes iguais de processos.

Exemplo: Se hoje os responsáveis são Milena, Sandra e Leandro, o 1º documento vai para Milena, o 2º para Sandra, o 3º para Leandro, o 4º volta para Milena. Ninguém recebe o segundo caso antes que todos tenham recebido o primeiro.

3. BLOQUEIO DE SOBRECARGA
Fica terminantemente proibido o sistema atribuir mais de 2 casos seguidos para o mesmo CPF se houver outros conselheiros ativos na escala do dia com menos casos atribuídos.
